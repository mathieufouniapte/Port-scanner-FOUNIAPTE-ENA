# Scanner de Ports TCP - Projet RSI4

## Présentation
Ce projet est un scanner de ports TCP simple et efficace développé en **Python**. Il permet de vérifier l'état des ports (ouvert/fermé) sur une adresse IP cible et d'identifier les services associés.

## Fonctionnalités
- Validation de l'adresse IP saisie.
- Scan d'une plage de ports définie par l'utilisateur.
- Identification du nom du service (ex: HTTP, SSH, FTP).
- Affichage des résultats en temps réel.

## Installation et Utilisation
1. Assurez-vous d'avoir Python installé (version 3.x).
2. Clonez ce dépôt ou téléchargez les fichiers.
3. Lancez le script principal :
   ```bash
   python main.py
